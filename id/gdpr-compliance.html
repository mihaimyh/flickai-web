Kepatuhan GDPR

Terakhir Diperbarui: 28 November 2025

1. Hak Anda

Berdasarkan Peraturan Perlindungan Data Umum (GDPR), Anda memiliki hak-hak berikut mengenai data pribadi Anda:

  • Hak untuk mengakses data pribadi Anda.
  • Hak untuk perbaikan data yang tidak akurat.
  • Hak untuk penghapusan ("hak untuk dilupakan").
  • Hak untuk membatasi pemrosesan.
  • Hak portabilitas data.
  • Hak untuk menolak pemrosesan.

2. Pemrosesan Data

Kami memproses data Anda secara sah, adil, dan transparan. Kami hanya mengumpulkan data untuk tujuan yang ditentukan, eksplisit, dan sah.

3. Hubungi Petugas Perlindungan Data

Untuk menggunakan hak Anda atau untuk pertanyaan mengenai kepatuhan GDPR, silakan hubungi Petugas Perlindungan Data kami di: support@flickai.net
